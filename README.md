# Contacts Book   

[Simple application](https://stormy-fjord-45497.herokuapp.com/) for adding, deleting and editing contacts.

## Install

```sh
$ make install
```

## Run

```sh
$ make start
# open http://localhost:5000
```
## Used   
   
**admin** - username and password.  
   
## Libraries used  

* React   
* Redux Toolkit   
* Formik   
* React Router Dom   
* SCSS   
* Soket.io   
* i18next
