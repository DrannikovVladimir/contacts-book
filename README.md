# Contacts Book   

[Simple application](https://stormy-fjord-45497.herokuapp.com/) for adding, deleting and editing contacts.

## Install

```sh
$ make install
```

## Run

```sh
$ make start
# open http://localhost:5000
```
## Used   
   
To enter the application username and password - **admin**
